## 方法一：diff 为不存在重复的两个元素异或的结果

```java

public void FindNumsAppearOnce(int[] array, int num1[], int num2[]) {
    int diff = 0;
    for (int a : array) {
        diff ^= a;
    }
    diff &= -diff;  //注意，得到这两个元素的最低位
    for (int a : array) {
        if ((diff & a) == 0) {
            num1[0] ^= a;
        } else {
            num2[0] ^= a;
        }
    }
}

```
