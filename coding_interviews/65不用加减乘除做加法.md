## 方法一：^ & <<，分别表示每位求和与计算进位值

```java

public int Add(int num1, int num2) {
    while (num2 != 0) {  //注意
        int tmp = num1 ^ num2;
        num2 = (num1 & num2) << 1;
        num1 = tmp;
    }
    return num1;
}

```
