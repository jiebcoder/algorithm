## 方法一：迭代，头插法 

```java

public ListNode ReverseList(ListNode head) {
    ListNode dummy = new ListNode(-1);
    while (head != null) {
        ListNode next = head.next;
        head.next = dummy.next;
        dummy.next = head;
        head = next;
    }
    return dummy.next;
}

```

## 方法二：递归

```java

public ListNode ReverseList(ListNode head) {
    if (head == null || head.next == null) {
        return head;
    }
    ListNode next = head.next;
    head.next = null;  //注意
    ListNode newHead = ReverseList(next);  //注意
    next.next = head;  //注意
    return newHead;
}
```
