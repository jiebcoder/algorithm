## 方法一：中序遍历，先右子节点，再父节点
   
```java

public TreeLinkNode GetNext(TreeLinkNode pNode) {
   if (pNode == null) {
       return null;
   }
   if (pNode.right != null) {
       pNode = pNode.right;
       while (pNode.left != null) {
           pNode = pNode.left;
       }
       return pNode;
   }
   while (pNode.next != null) {
       TreeLinkNode parent = pNode.next;
       if (parent.left == pNode) {
           return parent;
       }
       pNode = pNode.next;
   }
   return null;
}
```
